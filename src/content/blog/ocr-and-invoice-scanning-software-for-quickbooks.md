## OCR Technology for Invoice Scanning

Modern invoice OCR combines multiple technologies:

**Optical Character Recognition (OCR):** Converts images of text into machine-readable characters. Basic OCR extracts raw text but does not understand document structure.

**Machine Learning (ML):** Models trained on thousands of invoices understand document structure, recognize different layouts, and handle terminology variations.

**Natural Language Processing (NLP):** Helps the system understand context - distinguishing ship-to from bill-to addresses, identifying vendor vs customer names, and handling multiple languages.

**Vision + LLM Hybrid:** FormX uses proprietary image pre-processing and OCR result post-processing, combined with machine learning models trained specifically on financial documents. This achieves 92% extraction accuracy.
